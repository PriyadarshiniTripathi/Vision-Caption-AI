# Vision Caption AI
This project generates captions for images using a pretrained transformer model.

## Features
- Upload single or multiple images
- Generate captions automatically
- Supports multiple captions per image

## Technologies Used
- Python
- Transformers (BLIP model)
- PyTorch
- Google Colab

## How it works
1. Upload image
2. Image is processed into tensor format
3. Model generates caption
4. Output is decoded into text

## Future Improvements
- Batch processing for large datasets
- Web app deployment using Streamlit
